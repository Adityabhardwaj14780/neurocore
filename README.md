# NeuroCore

AI-powered full-stack platform with authentication, API orchestration, and scalable backend architecture.

---

## Features

- User Authentication System
- JWT-based Authorization
- Protected Routes & Middleware
- REST API Architecture
- MongoDB Database Integration
- AI Service Integration
- Dashboard System
- Modular Backend Structure
- Full-stack TypeScript Setup

---

## Tech Stack

### Frontend
- Vite
- React
- TypeScript

### Backend
- Node.js
- Express.js
- MongoDB

### Authentication & Security
- JWT Authentication
- Cookie-based Sessions
- Rate Limiting Middleware

## Installation

### Clone Repository

```bash
git remote add origin https://github.com/Adityabhardwaj14780/neurocore.git
cd neurocore
```

### Frontend Setup

```bash
npm install
npm run dev
```

### Backend Setup

```bash
cd backend
npm install
npm run dev
```
